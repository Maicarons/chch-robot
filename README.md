# chch-robot

> 注意: Pikafish的可执行文件请自行下载并放置在项目根目录下（默认引擎：./Pikafish/pikafish-avx2.exe，如需更改请到config.py中修改）。

## 项目概述

本项目实现了一个完整的线下中国象棋人机对弈系统，整合了：
- 视觉识别（OpenCV 摄像头拍摄棋盘）
- AI 引擎（Pikafish，基于 UCI 协议）
- 机械臂控制（执行 AI 走法）

## 系统架构

```
┌─────────────┐      ┌──────────────┐      ┌─────────────┐
│  摄像头拍摄  │ ──→  │  OpenCV 识别  │ ──→  │  FEN 生成   │
└─────────────┘      └──────────────┘      └──────────────┘
                                                  ↓
┌─────────────┐      ┌──────────────┐      ┌─────────────┐
│  机械臂执行  │ ←─ ─  │  UCI 走法解析  │ ←─ ─  │ Pikafish AI │
└─────────────┘      └──────────────┘      └──────────────┘
```

## 完整流程

### 1. 初始化和棋盘识别
```bash
python main.py
```

**步骤：**
1. 启动摄像头，拍摄初始棋盘
2. OpenCV 处理图像，识别每个棋子位置
3. 转换为 FEN 格式（第一步记录）

### 2. AI 决策
1. 发送 FEN 到 Pikafish 引擎（通过 UCI 协议）
2. Pikafish 思考并返回最佳走法（UCI 格式）
3. 更新棋盘 FEN（第二步记录）

### 3. 机械臂执行
1. 解析 UCI 走法（如 "h3e3"）
2. 转换为机械臂坐标
3. 控制机械臂移动棋子

### 4. 玩家操作和循环
1. 等待玩家走棋
2. 摄像头检测棋盘变化
3. 验证走法合法性
4. 返回步骤 2

## 核心模块说明

### 1. board_recognition.py - 棋盘识别
- 摄像头校准和图像捕获
- 棋盘格检测和定位
- 棋子识别（颜色、类型）
- FEN 串生成

### 2. ai_engine.py - AI 引擎通信
- UCI 协议实现
- Pikafish 引擎进程管理
- 走法获取和位置更新
- 思考时间和深度控制

### 3. robot_control.py - 机械臂控制
- 坐标变换（棋盘坐标 → 机械臂坐标）
- 路径规划
- 抓取和放置控制
- 安全保护

### 4. game_manager.py - 游戏管理
- 游戏状态管理
- 走法历史记录
- 规则验证
- 游戏结束检测

## 硬件需求

1. **摄像头**: USB 摄像头或网络摄像头（720p 以上）
2. **机械臂**: 6 自由度机械臂（如 DOBOT、大象机器人等）
3. **计算设备**: 运行 Windows/Linux 的 PC
4. **棋盘**: 标准中国象棋棋盘（建议固定位置和角度）

## 软件依赖

- Python 3.8+
- OpenCV 4.5+
- NumPy
- pyffish（可选，用于走法验证）

安装依赖：
```bash
pip install -r requirements.txt
```

## 配置说明

编辑 `config.py` 配置以下参数：

```python
# 摄像头配置
CAMERA_INDEX = 0
CAMERA_WIDTH = 1280
CAMERA_HEIGHT = 720

# 棋盘配置
BOARD_ROWS = 10
BOARD_COLS = 9
SQUARE_SIZE = 50  # 毫米

# AI 引擎配置
ENGINE_PATH = "./pikafish-avx2.exe"
ENGINE_DEPTH = 15
THINK_TIME = 5000  # 毫秒

# 机械臂配置
ROBOT_TYPE = "dobot"  # 或 "elephant_robotics"
HOME_POSITION = (100, 100, 150)  #  home 点坐标
GRIPPER_HEIGHT = 80  # 夹爪高度
```

## 坐标系说明

### 棋盘坐标系
- 文件（纵线）: a-i（从左到右，从红方视角）
- 横线：1-10（从红方底线到黑方底线）

### 机械臂坐标系
- X 轴：水平方向（对应棋盘文件）
- Y 轴：垂直方向（对应棋盘横线）
- Z 轴：高度（抓取/放置）

## 故障排除

### 棋盘识别失败
- 检查光照条件
- 重新校准摄像头角度
- 确保棋盘在视野中心

### AI 引擎无响应
- 检查引擎路径是否正确
- 确认引擎权限
- 查看引擎日志输出

### 机械臂移动异常
- 检查坐标系校准
- 验证目标位置是否可达
- 调整运动速度和安全高度
